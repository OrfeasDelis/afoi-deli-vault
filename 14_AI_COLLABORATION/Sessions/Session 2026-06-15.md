## 3. The afoideli.gr cross-reference — method and its honest limits

Goal: for each Kronos collection, record whether our own site carries it and at what URL — the start of a "cross-reference machine" (supplier collections vs. our range).

- afoideli.gr is a **JavaScript WooCommerce store**. `web_fetch` sees only pre-render HTML; on-site search needs JS; `web_search` finds only indexed pages and is polluted ("Kronos" → Greek titan, a death-metal band, a gyro company).
- **Confirmed present** (real product pages surfaced): **Nativa** (`/product/nativa/`, "Nativa Kronos"), **Woodside** (`/product/woodside/`), **Prima Materia** (flip-book `/3d-flip-book/prima-materia-kronos/`).
- The other 12 are marked **`unverified — confirm via Store API`**, NOT "absent." Deliberate: false negatives would poison the cross-reference machine. A prominent warning callout in the reference note explains why.
- **The robust path (chosen by Orfeas):** the WooCommerce **Store API** — `afoideli.gr/wp-json/wc/store/v1/products?search=<name>` — unauthenticated, returns clean JSON with `permalink`. Couldn't be called from Claude's side (hosted `web_fetch` is blocked from arbitrary `/wp-json/` URLs not already in a search result). **Must run from Orfeas's side** (Python worker / curl / paste JSON back). This is the next concrete step to finish the column.
- **Match rule (chosen):** flag fuzzy matches as "possible — verify, with both names"; record exact matches; never silently equate a renamed product.

## 4. Decisions

- **Verification engine = WooCommerce Store API**, run operator-side. Search-only is last resort and its negatives are always `unverified`, never `absent`.
- **Folder-per-supplier** across the whole `Suppliers/` tree (not just Kronos), so future child notes (collection refs, asset plans, project links) have a home.
- **Telegram capture queue before the monthly auto-diff.** You are the trigger and the judgment; Hermes records and fetches. The auto-diff becomes a second feeder into the same queue later. Honours the interface doctrine: the queue (file/table) is the source of truth, not the agent.
- **Reuse the [[Circles]] Telegram bot host** for the `/add` command + file handler — don't stand up a second bot.
- **Link-back pattern generalises:** binary in Drive, link+metadata in vault. Price lists ride the same rails (`_pricelists/` → the supplier note's "Price list folder" field).

## 6. Method note — search vs. JS sites

Reinforced limit for future sessions: for any **JavaScript-rendered site** (WooCommerce, most modern storefronts), neither `web_fetch` nor `web_search` is authoritative. Use the site's **JSON API** if it has one (WordPress/WooCommerce expose `/wp-json/`), or a real browser (Claude in Chrome) as fallback. Record uncertain absences as `unverified`, never as fact.
