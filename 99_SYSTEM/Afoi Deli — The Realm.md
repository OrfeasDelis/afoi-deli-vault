## How to read it

- **The light is the truth.** Colour encodes build state — crimson = built · gold = in progress · cyan = specified, not yet lit · grey = concept. Most of the sky is dark *on purpose*: only Layer 1 (Obsidian) actually runs; the schemas hold zero rows. The unlit void is the unbuilt OS.
- **❤ The Heart** (center) — the source: the three nested layers (merchant → steward → the man to be looked up to), the two maxims, the family beneath. → [[The Heart]] · [[Afoi Deli — Operating Doctrine]]
- **The worlds** — Supplier Highlands (04), Order Harbor (02), the Showroom (05), Finance Keep (10), the Atelier Library (07), Projects Field (06), the Expansion Frontier (11), and the distant Personal Vale (12·15·16·17, with the Journal as a moon — *author by invitation*). Folder index: [[Vault Map]].
- **The comet** — the order lifecycle: quote → order 0→1 → PO/proforma → received/delivered → invoiced/paid. The spine: [[Order Workflow 0-4]] · [[Order Lifecycle — Ground-Truth Capture]].
- **The orbit-rings** — the 5-layer OS (Knowledge → Data → Automation → Interface → Agent). Where the build actually stands: [[Vault State Memory]].
