# Quote Creation Checklist

## Before sending quote

- [ ] Client/project name clear
- [ ] Supplier clear per line
- [ ] Product code correct
- [ ] Description clear
- [ ] Finish/size/color confirmed
- [ ] Unit correct
- [ ] Quantity correct
- [ ] Packaging checked where needed
- [ ] Selling price correct
- [ ] Discount approved
- [ ] VAT clear
- [ ] Delivery/transport excluded or included
- [ ] Installation excluded or included
- [ ] Lead time communicated carefully
- [ ] Validity period stated

## After quote acceptance

- [ ] Deposit/payment status confirmed
- [ ] Order file saved to Daily
- [ ] All lines status 0
- [ ] Sales notes included
- [ ] Special instructions visible
- [ ] Architect/client reference included
