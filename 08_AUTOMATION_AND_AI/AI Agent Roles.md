# AI Agent Roles

## 1. Operations Copilot

Reads:

- Kouvas snapshots
- Order notes
- Supplier notes
- SOPs

Helps with:

- what is stuck
- who needs follow-up
- what changed today
- what folder an order belongs to
- which exception applies

## 2. Proforma Checker

Reads:

- supplier proforma
- order lines
- product records
- supplier price rules

Flags:

- code mismatch
- quantity mismatch
- price mismatch
- missing lines
- unexpected terms

## 3. Supplier Intelligence Agent

Reads:

- supplier emails
- supplier notes
- catalogues
- price lists

Outputs:

- lead time patterns
- relationship issues
- common errors
- best products
- strategic recommendations

## 4. Website Content Agent

Reads:

- product knowledge
- brand notes
- project cases
- SEO map

Creates:

- collection page drafts
- journal articles
- social captions
- moodboard texts
- architect guides

## 5. Personal Chief of Staff

Reads:

- personal OS
- priorities
- weekly reviews
- calendar/tasks if connected later

Helps with:

- focus
- routines
- health discipline
- weekly planning
- high-leverage decisions
