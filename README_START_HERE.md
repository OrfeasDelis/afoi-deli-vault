# AFOI DELI SECOND BRAIN — START HERE

This vault is designed as the **operating infrastructure** for your business, your future ventures, and your personal command system.

It is not just for storing notes.  
It is built so that later you can connect it to:

- n8n automations
- Gmail parsing
- Excel / Kouvas migration
- supplier and product databases
- AI agents over company knowledge
- website content systems
- sales and order dashboards
- future ERP-like interfaces

## The core rule

> Document first. Structure second. Automate third.

Do not automate chaos.  
Every workflow, rule, exception, supplier detail, client note, product code, and decision should eventually live here in a clean, findable, machine-readable form.

## Vault philosophy

This vault is built around four permanent layers:

### 1. Memory
Facts, projects, suppliers, clients, products, pricing rules, procedures.

### 2. Control
Dashboards, priorities, decision logs, KPIs, risks, alerts.

### 3. Automation
Schemas, clean frontmatter, repeatable SOPs, n8n-ready workflows.

### 4. Creation
Website content, brand material, case studies, future ventures, personal thought leadership.

At the end of every business day, add only three things:

1. One process detail you noticed.
2. One problem or exception.
3. One idea for automation.
