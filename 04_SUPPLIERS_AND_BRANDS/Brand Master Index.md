# Brand Master Index

## Purpose

Brands are not only supplier names.  
They are taste, positioning, application, technical reality, and sales language.

## Brand note naming

Use:

```text
Brand - Brand Name
```

## Brand note should include

- What the brand stands for
- Best categories
- Best use cases
- Price level
- Lead time reality
- Target client
- Architect appeal
- Showroom value
- Website/content potential
- Technical risks
- Alternatives
